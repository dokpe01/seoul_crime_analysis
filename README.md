## **프로젝트 개요**
<br/>

> ### CPTED란?<br/>
> "Crime Prevention Through Environmental Design"의 약자로, 환경 설계를 통한 범죄 예방을 의미한다.
<br/>

- 범죄예방 이론 및 정책 중에서도 CPTED의 범죄예방 효과에 주목하여, 이를 기반으로 한 통계적 분석과 그 결과를 통한 안전지도를 구현한다.
<br/>

- 또한 CPTED가 과연 범죄 억제 장치의 기능을 실질적으로 보여주는 지를 데이터 기반의 결과로 나타내는 것을 최종 목표로 한다.
<br/>

## 분석 문항
1. 서울특별시 자치구별 범죄율 분석 및 시각화
2. 인구가 밀집할수록 범죄 발생 빈도도 증가하는가?
3. 범죄가 자주 발생하는 특정 시간대/요일이 있는가?
4. CCTV 수가 증가하면 범죄 발생 빈도가 감소하는가?
5. 안전비상벨이 증가하면 범죄 발생 빈도가 감소하는가?
6. 파출소 수/순찰 인력이 많은 지역은 범죄 발생율이 낮은가?
7. 근린시설 수가 많은 구역은 범죄 발생율이 낮은가?
8. 여성안심지킴이집은 범죄예방에 효과가 있는가?
9. 유흥업소(단란주점)이 밀집한 지역은 범죄 발생율이 높은가?
10. 유동인구가 많은 지역은 범죄 발생율이 높은가?

- 특정 요인이 어떤 범죄와의 어떤 관계가 있는지 통계적 분석을 시도한다.
